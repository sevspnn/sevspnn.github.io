# Severino Netto — Portfólio Acadêmico e Profissional

Repositório do site pessoal e portfólio acadêmico de **Severino Peixoto Nunes Netto**, com informações sobre trajetória profissional, projetos de pesquisa, publicações, apresentações e materiais didáticos.

## 🌐 Acesse o site

**Site publicado:** https://sevspnn.github.io/

## 📌 Sobre o projeto

Este repositório contém o código-fonte do site estático utilizado como portfólio pessoal e acadêmico.  
A página foi desenvolvida para apresentar, de forma clara e organizada:

- trajetória profissional e acadêmica
- projetos de pesquisa e desenvolvimento
- publicações e apresentações
- materiais didáticos de disciplinas
- formas de contato e redes profissionais

O site também funciona como espaço de divulgação científica e educacional, reunindo conteúdos ligados à Engenharia Biomédica, Robótica Macia, Jogos Sérios, Tecnologias Assistivas, Processamento de Sinais Biológicos e Inteligência Artificial.

## 🧩 Seções do site

O portfólio está organizado nas seguintes áreas:

- **Sobre**  
  Apresentação profissional, formação, áreas de interesse e links rápidos.

- **Projetos**  
  Destaque para projetos de pesquisa e desenvolvimento, como:
  - SighGame — Reabilitação Respiratória Pediátrica
  - Métodos de Elementos Finitos em Robótica Macia
  - IA e Processamento em Sinais Biológicos

- **Publicações e Apresentações**  
  Produção científica, trabalhos apresentados em eventos e revisões em desenvolvimento.

- **Material Didático**  
  Materiais de disciplinas, incluindo aulas em PDF, provas, listas de exercícios e links suplementares.

- **Contato**  
  Links para redes profissionais e canais de comunicação.

## 🛠️ Tecnologias utilizadas

Este projeto foi construído como um site estático, com foco em simplicidade, leveza e fácil manutenção.

- HTML
- GitHub Pages
- organização enxuta para hospedagem estática
- suporte a tema claro/escuro
